# index_signin
第一个完整点的小项目,前端html+css+js,后端Django,数据库MySQL,特此纪念

## 使用技术
前端HTML+css+js,  
后端Django（Python）  
数据库MySQL 
页⾯插件：⾳乐播放器：APlyer，提⽰框：sweetalert 

## 功能:
①注册（正则表达式检验注册格式，后端检验是否已被注册）   
②登录:登录成功后获得cookie（下次进⼊⽹页则不需要再登录）   
③主页:若已登录，则右上⾓显⽰个⼈昵称； 主页有基本该有的布局及内容，还加⼊了⽹页⾳乐播放器
